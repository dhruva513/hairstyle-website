# ✂️ StyleFinder - Hairstyle Recommender

A beautiful web application that recommends the perfect hairstyle 
based on your face shape. Built with Python Flask backend and 
modern HTML/CSS/JavaScript frontend.

![Python](https://img.shields.io/badge/Python-3.10-blue)
![Flask](https://img.shields.io/badge/Flask-3.0-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

## 🌟 Features

- 7 Face Shapes: Oval, Round, Square, Heart, Oblong, Diamond, Triangle
- 84+ Hairstyle Recommendations
- Gender Toggle: Separate recommendations for Men and Women
- Celebrity Inspiration for each face shape
- Styles to Avoid section
- Beautiful dark theme UI with animations
- Fully responsive design
- Python Flask backend with REST API
- Search functionality
